1 1. 概述

本文参考了 Ubuntu 中文 Wiki。

2 2. 更新步骤

2.1 2.1. 首先备份源列表：

sudo cp /etc/apt/sources.list /etc/apt/sources.list_backup

2.2 2.2. 而后用 gedit 或其他编辑器打开：

gksu gedit /etc/apt/sources.lis

2.3 2.3.

从下面"Ubuntu13.10 源列表"中选择合适的源，替换掉文件中所有的内容，保存编辑好的文件，注意一定要选对版本：

2.4 2.4. 刷新列表，注意：一定要执行刷新：

sudo apt-get update

2.5 2.5. 更新系统（可选）：

sudo apt-get upgrade

3 3. 使用 "apt-fast" 更新提速

3.1 3.1. 安装 "apt-fast"：

sudo apt-get install axel aria2
sudo add-apt-repository ppa:apt-fast/stable
sudo apt-get update
sudo apt-get install apt-fast

3.2 3.2. 对 "apt-fast" 进行设置（新版本安装完成后会显示设置窗口）：

sudo gedit /etc/axelrc

# 设置合适的线程数，默认为 4
num_connections = Num

3.3 3.3. 用 "apt-fast" 代替 "apt-get" 使用：

sudo apt-fast update
sudo apt-fast install **
sudo apt-fast upgrade
